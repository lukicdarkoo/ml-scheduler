# Task Scheduler based on Machine Learning

## Instalation
Make sure that you have installed Python & pip:
- Arch `pacman -S python pip`
- Debian `apt-get install python pip`

```python setup.py install```

## Run
```python mlscheduler```

## Testing
```python -m unittest test.TaskTest```
